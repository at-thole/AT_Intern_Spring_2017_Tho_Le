####What is HTML ?
HTML is Hyper Text Markup Language, the standard markup language for creating Web pages.

A one page html basic :
```python
<!DOCTYPE html>
<html>
<head>
  <title></title>
</head>
<body>

</body>
</html>
```

####Struct of html page
HTML can be divided into different parts
```
1.Head: <title>, <meta>, <link> 
2.Body: <header>, <nav>, <section>, <aside>, <article>, <figure>, <footer>, <a>, <p>, <h>
...
```
